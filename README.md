# Rust-practice

# Installation
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
```
rustc --version
```
**compiling rust program** 
```
rustc <file-name>
```

**Running rust files**
```
./<file-name>
```
# Using cargo

**Creating new files**
```
cargo new <file-name>
cd <file-name>
```
**Compiling**
```
cargo build
```
**Running**
```
cargo run
```
